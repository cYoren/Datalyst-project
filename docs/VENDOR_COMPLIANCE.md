# Vendor Compliance Checklist

Maintain this list for LGPD/GDPR accountability. Fill in provider names, data locations, and signed DPAs.

## Processors
- Supabase (auth, database)
- Vercel (hosting)
- Email provider (transactional email)
- Analytics provider (if added)

## Required Artifacts
- Data Processing Agreements (DPAs) signed with each processor.
- List of subprocessors and data locations.
- International transfer safeguards (e.g., SCCs) if data leaves Brazil or the EEA.
- Record of Processing Activities (ROPA) covering purposes and data categories.

## Review Cadence
- Review vendor list quarterly.
- Re-assess transfers when adding new tools or regions.
